The ``1-my_list`` module
========================

Using MyList
------------

Import MyList:

    >>> MyList = __import__('1-my_list').MyList

Test with append and print_sorted:

    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test print_sorted with an already sorted list:

    >>> sorted_list = MyList()
    >>> sorted_list.append(1)
    >>> sorted_list.append(2)
    >>> sorted_list.append(3)
    >>> sorted_list.print_sorted()
    [1, 2, 3]

Test print_sorted with negative numbers:

    >>> neg_list = MyList()

    >>> neg_list.append(-5)
    >>> neg_list.append(3)
    >>> neg_list.append(0)

Test print_sorted with an empty list:
