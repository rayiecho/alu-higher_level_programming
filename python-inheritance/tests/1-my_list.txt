The ``1-my_list`` module
========================

Using MyList
------------

Import MyList:

    >>> MyList = __import__('1-my_list').MyList

Check MyList inherits from list:

    >>> issubclass(MyList, list)
    True

Check instantiation:

    >>> my_list = MyList()
    >>> isinstance(my_list, MyList)
    True

Check append:

    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Check print_sorted with sorted append:

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Check the original list is not sorted (print_sorted doesn't modify it):

    >>> print(my_list)
    [1, 4, 2, 3, 5]

Check print_sorted with an already sorted list:

    >>> sorted_list = MyList()
    >>> sorted_list.append(1)
    >>> sorted_list.append(2)
    >>> sorted_list.append(3)
    >>> sorted_list.print_sorted()
    [1, 2, 3]

Check print_sorted with negative numbers:

    >>> neg_list = MyList()
    >>> neg_list.append(-1)
    >>> neg_list.append(-5)
    >>> neg_list.append(3)
    >>> neg_list.append(0)
    >>> neg_list.print_sorted()
    [-5, -1, 0, 3]

Check print_sorted with an empty list:

    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Check print_sorted returns None (only prints):

    >>> empty_list.print_sorted() is None
    []
    True

Check print_sorted returns a new list (original untouched):

    >>> mixed = MyList()
    >>> mixed.append(9)
    >>> mixed.append(1)
    >>> before = list(mixed)
    >>> mixed.print_sorted()
    [1, 9]
    >>> list(mixed) == before
    True
