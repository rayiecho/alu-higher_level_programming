# python-data_structures

This project covers the basics of Python data structures: lists, tuples,
and their manipulation without relying on built-in helper functions where
restricted by the task requirements.

## Requirements

- Ubuntu 20.04 LTS, python3 (3.8.5)
- Files styled per `pycodestyle` (2.7.*)
- Every file starts with `#!/usr/bin/python3`, ends with a newline, and is
  executable
- No `import` statements unless a task explicitly allows it

## Tasks

| # | File | Description |
|---|------|-------------|
| 0 | `0-print_list_integer.py` | Print all integers of a list |
| 1 | `1-element_at.py` | Safely access an element of a list by index |
| 2 | `2-replace_in_list.py` | Replace an element of a list in place |
| 3 | `3-print_reversed_list_integer.py` | Print all integers of a list in reverse |
| 4 | `4-new_in_list.py` | Replace an element in a copy of a list |
| 5 | `5-no_c.py` | Remove all `c`/`C` characters from a string |
| 6 | `6-print_matrix_integer.py` | Print a matrix of integers |
| 7 | `7-add_tuple.py` | Add two tuples element-wise |
| 8 | `8-multiple_returns.py` | Return length and first character of a string |
| 9 | `9-max_integer.py` | Find the biggest integer in a list |
| 10 | `10-divisible_by_2.py` | Find all multiples of 2 in a list |
| 11 | `11-delete_at.py` | Delete an item at a specific position |
| 12 | `12-switch.py` | Swap two variables in 5 lines |
