# python-exceptions

Project on Python exceptions: try/except/finally, raising exceptions,
and safe printing functions that handle bad input gracefully.

## Tasks

| # | File | Description |
|---|------|-------------|
| 0 | 0-safe_print_list.py | Print x elements of a list |
| 1 | 1-safe_print_integer.py | Safely print an integer |
| 2 | 2-safe_print_list_integers.py | Print and count integers in a list |
| 3 | 3-safe_print_division.py | Divide two integers with debug output |
| 4 | 4-list_division.py | Divide two lists element by element |
| 5 | 5-raise_exception.py | Raise a type exception |
| 6 | 6-raise_exception_msg.py | Raise a name exception with a message |
