# python-more_data_structures

Second data structures project: squaring matrices, search/replace, sets,
dictionaries, map(), and a Roman numeral converter.

## Tasks

| # | File | Description |
|---|------|-------------|
| 0 | 0-square_matrix_simple.py | Square every value of a matrix |
| 1 | 1-search_replace.py | Replace all occurrences of an element |
| 2 | 2-uniq_add.py | Sum all unique integers in a list |
| 3 | 3-common_elements.py | Common elements between two sets |
| 4 | 4-only_diff_elements.py | Elements present in only one set |
| 5 | 5-number_keys.py | Number of keys in a dictionary |
| 6 | 6-print_sorted_dictionary.py | Print dictionary sorted by key |
| 7 | 7-update_dictionary.py | Replace or add a key/value pair |
| 8 | 8-simple_delete.py | Delete a key from a dictionary |
| 9 | 9-multiply_by_2.py | Multiply all dictionary values by 2 |
| 10 | 10-best_score.py | Key with the highest value |
| 11 | 11-multiply_list_map.py | Multiply list values using map() |
| 12 | 12-roman_to_int.py | Convert a Roman numeral to an integer |
